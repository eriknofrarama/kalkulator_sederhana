# Kalkulator Sederhana

Aplikasi kalkulator sederhana menggunakan **HTML, CSS Grid, dan JavaScript**.

## Cara Menjalankan
1. Clone repo ini atau buka langsung melalui GitHub Pages.
2. Buka file `index.html` di browser.
